# css-selectors

[CSS 选择符概览](https://yanxyz.github.io/css-selectors)

## 说明

- `docs` 目录是 github pages
- `src` 目录是源码，编辑 `data.yaml` 之后运行 `node build.js` 将它转为 HTML。
- `spec` 目录，`node check.js` 比较源文档。
